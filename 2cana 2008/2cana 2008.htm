Question    
Details of a lethal bio weapon called “Anthrobola”, a deadly mix between Anthrax and the Ebola virus, have been given to us.
The perpetrators have stated that if their demands for 1 bajillion dollars (USD) are not met within 4 hours, they will deploy the bio weapon in a densely populated area.
The leader of the organisation claims himself to be the next Robert Jung. His sinister plot needs to be foiled at all costs!
In order to prevent detonation of the weapon, you need to decrypt the information leaked to us by our spies.
Refer to guidelines below on how the Decryption Machine would function.
We have 3 segments of information.
The first segment, we believe, is part of the URL used to infiltrate their system.

The second is the location of the weapon.

The third is the code used to disarm the weapon.

The three actual code segments will be revealed by our spies shortly - we estimate in approximately one hour

Guidelines to designing the Decryption Machine

Stage One

Rules:

- Each string has alphabetic characters embedded in it.
- After each alphabetic character is a range of operations to be applied to the preceding alphabetic character.
- A + denotes that the deciphered character is actually the next character in the alphabetic sequence, wrt the initial character.
- A - denotes that the deciphered character is actually the previous character in the alphabetic sequence, wrt the initial character.
- The string is case-insensitive.
- Spaces remain as spaces.

E.g.
Given the string “c+++++w++++f--+--k-+--+-++ j-m+ s++--+f-b-l+q++”
We can decipher this as follows:
c+++++ =     h(c+++++ = d++++ = e+++ = f++ = g+ = h)
w++++ =      a (w++++ = x+++ = y++ = z+ = a)
f--+-- =      c (f--+-- = e-+-- = d+-- = e-- = d- = c)
k-+--+-++ =   k (k-+--+-++ = j+--+-++ = k--+-++ = j-+-++ = i+-++ = j-++ = i++ = j+ = k)
space
j- =      i (j- = i)
m+ =      n (m+ = n)
space
s++--+ =    t (s++--+ = t+--+ = u--+ = t-+ = s+ = t)
f- =      e (f- = e)
b- =      a (b- = a)
l+ =      m (l+ = m)
q++ =     s (q++ = r+ = s)
We decipher this to be the string “hack in teams”

Stage Two
Introduce the # operation into the encrypted string.
Amendment to Rules:
- The # is the “inverse” of the initial letter. (ie: a# = z, b# = y etc.)
E.g.
Given the string “c+++++#w+#++++++f--+--k-+--+-++”
We can decipher this as follows:
c+++++# =      s (c+++++# = d++++# = e+++# = f++# = g+# = h# = s)
w+#++++++ =      i (w+#++++++ = x#++++++ = c++++++ = d+++++ = e++++ = f+++ = g++ = h+ = i)
f--+-- =      c (f--+-- = e-+-- = d+-- = e-- = d- = c)
k-+--+-++ =      k (k-+--+-++ = j+--+-++ = k--+-++ = j-+-++ = i+-++ = j-++ = i++ = j+ = k)
We decipher this to be the string “sick”

Stage Three

Introduce single digit (1 to 9) numbers next to each operation.
Amendment to Rules:
- A number implies that the preceding manipulation is applied that many times.

E.g.
Given the string “c+5#w+#+2+3+f-2+--k-+--+-++”
We can decipher this as follows:
c+5# =      s (c+5# = c+++++# = d++++# = e+++# = f++# = g+# = h# = s)
w+#+2+3+ =      i (w+#+2+3+ = w+#++++++ = x#++++++ = c++++++ = d+++++ = e++++ = f+++ = g++ = h+ = i)
f-2+-- =      c (f-2+-- = f--+-- = e-+-- = d+-- = e-- = d- = c)
k-+--+-++ =      k (k-+--+-++ = j+--+-++ = k--+-++ = j-+-++ = i+-++ = j-++ = i++ = j+ = k)
We decipher this to be the string “sick”
